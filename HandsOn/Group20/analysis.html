Los datos serán utilizados para crear una guía de lugares donde salir por la noche de fiesta.

Data Analysis:

Terraza String Contiene un float que es la superficie, y dos integer para el numero de mesas y sillas permitidas respectivamente.
Local String Contiene el rotulo del local que es un string.
Distrito String Contiene el barrio, tipo de via, nombre de calle y portal en el que se encuentra un cierto local.
